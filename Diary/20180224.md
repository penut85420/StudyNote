### Max Heap

Max Heap 是一種 Full Binary Tree，是一種用 Array 表示的 Tree

當 Parent Node 的 Index 為 n 時，Child Node 的 Index 為 n \* 2 + 1 和 n \* 2 + 1

若 Index 從 1 開始則為 n \* 2 和 n \* 2 + 1

| Index | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 |
|:-----:|:-:|:-:|:-:|:-:|:-:|:-:|:-:|:-:|:-:|
| Value | 21| 17| 12|  6|  3|  1| - <td colspan=2>-|

其中，任何父節點的值恆大於子節點的值
